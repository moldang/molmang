'인터넷 vs 웹' 같을까?
인터넷이 도시라면 웹은 도시에 있는 빌딩 하나라고 볼 수 있다.
인터넷이 바다라면 웹은 바다로 흘러가는 하나의 강줄기라고 볼 수 있다.


인터넷이 작동하기 위한 조건
최소 2대의 컴푸터가 있어야 한다.
web browser와 web server가 있다.
web.info.ch라는 주소가 부여된 web server에는 index.html이라는 웹페이지가 있다.
web browser에서 web server의 web.info.ch/index.html을 검색하면 web server에서는 하드디스크에서 index.html에 해당하는 웹페이지를 찾아낸 후 다시 web browser에 신호를 답한다. 신호를 받은 web browser는 index.html을 코드를 맞게 해석하여 화면에 노출시킨다.
웹브라우저는 '정보'를 요청하며 웹서버는 요청에 대한 응답'을 하고 있다.
이를 우리의 자본주의 삶에 비유해서 표현하면 웹브라우저는 고객(client) 웹서버는 사업자(server)라고 부른다. 게임으로치면 우리가 설치하는 것은 game clinet이며 회사가 가지고 있는 것은 게임서버라고 한다.
